> 정치인·예산·공약을 시민이 쉽게 알 수 있는 책무성 플랫폼

## 2. 비전 / 문제 정의

### 문제

> "정치인들은 선거 때만 나오고, 시민은 무지한 상태로 투표한다."

- 후보·현직 의원 정보가 흩어져 있어 시민이 찾기 어렵다
- 의원들이 임기 중 무엇을 하는지 시민이 알기 어렵다
- 공약을 지키는지 검증할 도구가 없다
- 내 세금이 어디 쓰이는지 알기 어렵다
- 의정활동비·해외출장·이슈를 종합해서 보여주는 곳이 없다

### 비전

> **시민이 5분 안에 본인 지역구 정치인의 전체 책무성을 검증할 수 있는 사이트**

## ⭐ 핵심 페이지: 지역 허브 (Region Hub)

> "여주에 살면, 여주 한 페이지에서 의원·예산·회의록·후보 다 본다"

**URL:** `/region-hub?sido=경기도&wiwName=여주시`

**시민 흐름:**
1. 홈에서 "어디 사세요?" 한 번 입력 (시·도 → 시·군·구)
2. localStorage 저장 → 다시 묻지 않음
3. 지역 허브로 자동 이동
4. 한 페이지에서 모든 정보 스크롤

**섹션 (스크롤 순):**

| 섹션 | 내용 |
|---|---|
| 헤더 | 지역명 · 인구 · 2024 결산 총액 |
| 👥 우리 지역 의원 | 국회의원 (해당 선거구) · 도의원 · 시의원 · 시장 |
| 💰 우리 시 예산·결산 | 분야별 차트 + 부문 drill-down + 원본 PDF |
| 📋 회의록 (CLIK 후) | 최근 회의 + 의원별 발언 |
| 🗳️ 다가오는 선거 후보 (선거 시즌) | 시장 후보 + 시의원 후보 |
| 🔗 외부 링크 | 시청 · 시의회 · 도청 홈페이지 |

**전체 사이트 흐름:**
- 지역 입력은 한 번만
- 모든 페이지 상단에 "내 지역: 경기도 여주시 [변경]" 표시
- 의원·후보·예산 등 모든 페이지가 기본적으로 내 지역 데이터 우선 표시

## 4. 핵심 가치 명제

1. **로컬 우선** — 시·도·시·군·구 단위로 내 지역에 집중
2. **데이터 중심** — 공식 자료 출처 표기, 시민 의견과 분리
3. **비교·맥락** — 단순 숫자가 아닌 평균·전년 대비·이웃 지역 비교
4. **연결** — 의원 ↔ 법안 ↔ 표결 ↔ 예산 ↔ 발언 모두 hyperlink
5. **투명** — 출처와 갱신일 명시, 누락된 데이터도 솔직히 표시

## 5. 정보 구조 (IA) — 메뉴 재정리 제안

### 현재 nav (혼란)
홈 / 의원 / 후보 / 예산 / 제도 알아보기

### 제안 nav

```
🏠 홈 → 지역 입력 → 자동으로 지역 허브로
📍 내 지역 (지역 허브 — 본 사용자의 시·군·구)
👥 정치인 (전국 검색)
   ├─ 국회의원
   ├─ 광역의회 (시·도의원)
   ├─ 기초의회 (시·군·구의원)
   └─ 지방선거 후보 (선거 시즌만)
💰 예산·결산
   ├─ 국가 예산
   ├─ 광역 예산·결산
   └─ 시·군·구 결산
🔍 통합검색 (이름·법안·키워드)
📚 알아보기 (제도·용어·자주 묻는 질문)
```

### 페이지 계층

```
/                          홈 (지역 자동 감지 + 주요 카드)
/search                    통합검색 결과
/legislator/[id]           의원 상세
/candidate/[id]            후보 상세
/bill/[billId]             법안 상세
/region/[id]               지역구 단위 의원 모음
/budget                    예산 (3 탭)
/about                     제도 안내
```

## 7. 사용자 여정 (Top 3)

### 여정 1: "내 동네 의원 누구야"
1. 홈 진입
2. "내 지역 자동 감지" 또는 시·도 → 시·군·구 입력
3. 국회의원 / 광역의원 / 기초의원 카드 표시
4. 의원 클릭 → 프로필 + 재산 + 법안/표결 (국회의원) 또는 외부링크 (광역·기초)

### 여정 2: "내 세금 어디 쓰이나"
1. 홈에서 "예산 보기" 카드
2. 시·도 선택 → 결산 탭 → 시·군·구
3. 분야별 막대 → 클릭 → 부문별 drill-down
4. 원본 결산서 PDF 클릭

### 여정 3: "이번 지방선거 후보 누구"
1. 홈 → "지방선거" 카드
2. 시·도지사 / 시장군수구청장 라디오
3. 시·도 + 시·군·구 선택
4. 후보 카드 그리드
5. 후보 클릭 → 프로필 + (5/18 후) 5대 공약 + 전과·재산·병역·세금

## 10. 현재 UI 문제점 (재정리 대상)

1. **홈 카드 너무 분산** — 국회/광역/기초/지방선거 4 카드 + 예산 별도. 시민이 "내 지역" 한 번에 다 보고 싶음.
2. **시·도·시·군·구 입력 반복** — 각 페이지마다 다시 선택. 한 번 선택하면 세션 동안 유지되도록.
3. **광역·기초의원에 발의 법안 빈 탭** — 이미 숨김 처리됨 (✅)
4. **검색 기능 없음** — 이름·법안·키워드 입력 못 함
5. **외부 링크 분산** — likms·CLIK·각 의회 사이트가 페이지마다 다른 위치
6. **모바일 최적화 부족**
7. **출처·갱신일 표기 inconsistent**

## 11. UI 재정리 제안 (다음 단계)

### Phase 1: 홈 재설계
- Hero: "어디 사세요?" 단일 입력
- 입력 후 → 내 지역구 의원·후보·예산 통합 카드
- 세션 동안 지역 정보 localStorage 저장
- 보조: 통합검색 박스

### Phase 2: 의원 페이지 통일
- 국회/광역/기초 모두 같은 layout (있는 정보만 표시)
- 위쪽: 프로필 + 사진 + 정당 배지
- 중간: 활동 카드 (법안·표결·조례안·회의록 — 있는 것만)
- 아래: 재산·전과·병역·세금 (있는 것만)
- 우측 사이드: 외부 링크 모음

### Phase 3: 예산 단순화
- 탭 통일: 국가 / 시·도 / 시·군·구
- 항상 sticky 시·도 선택기

### Phase 4: 검색
- 상단 nav에 검색창
- 결과 페이지: 카테고리별 그룹 (의원·후보·법안·예산)
